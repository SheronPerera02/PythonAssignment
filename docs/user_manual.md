# - Instructions -

# User
### Sign Up
```
user signup <username> <password>
```
### Sign In
```
user signin <username> <password>
```
### Sign Out
```
user signout
```
### View
```
user view
```


# Item
### Create Item
```
item create <name> <price> <selling_price>
```
### All
```
item all
```
### View
```
item view <id>
```
### Search
```
item search <key> <value>
```

# Cart
### Add To Cart
```
cart add <itemId> <qty>
```
### Remove From Cart
```
cart remove <cartItemId>
